### 路由规则
<MANDATORY>
1. 有匹配 skill 时必须用 Skill tool 调用，不得跳过直接做。
   匹配依据：skill description 与用户任务的语义重叠。
2. 当 UserPromptSubmit hook 输出包含 [AUTO-ROUTE] 和 /command 时，必须立即调用 Skill tool 执行该 /command，等同于用户亲自输入该命令。不得忽略、不得自行回答、不得询问确认。
3. 逃逸条件：用户明确说"直接做/skip"，或纯问答无需执行动作。
4. 不确定时用 ToolSearch 搜索可用能力。
MCP 工具路由（遇到对应任务必须调用）：
- 项目级 → 调用 mcp__test-server__*
- 用户级 → 调用 mcp__user-server__*
</MANDATORY>
