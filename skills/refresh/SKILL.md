## 你的任务

1. **新增的能力** — 本次扫描出现、之前未见的 skills / agents / plugins / MCP servers

3. **无变化** — 若两次一致，简短说明"环境能力无变化"

如果本次是对话中的首次扫描（没有历史可对比），直接汇总当前能力概览即可。
